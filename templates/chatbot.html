This is a demonstration of a chatbot program written in Prolog. Currently there is no handling of sessions or state in the server, so the program doesn't remember earlier sentences.

The vocabulary is limited, so here is a list of words that can be used safely: dog, cat, human, park, school, fur, peter, lisa, johny, cute, little, brown, loud, do, have, who, what, where, when, how.

example sentence: "johny met a dog"
